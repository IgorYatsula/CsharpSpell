
CsharpSpell

Проект состоит из следующих частей:
1) консольная утилита (папка c исходниками: 'CsharpSpell')
2) тесты для проверки корректности алгоритма (папка с иходниками: 'CsharpSpell.Tests')

Также в корневой папке можно найти следующее:
1) папка 'CodeCoverage' - наглядный отчет по покрытию тестами кода
2) папка 'Dependencies' - "зависимости" проекта (сборки, которые были использованы в процессе реализации), использовались только для реализации и запуска тестов ('NuGet' не был использован нарочно)
3) 'bat' файлы для сбора проекта и запуска тестов: 'BuildRelease.bat', 'RunTests.bat'

Проект уже скомпилированный и бинарные файлы находятся на "дефолтном" месте: Bin/Debug и Bin/Release. 
Для удобства тестирования были созданы и размещены рядом с бинарным файлом "bat" файл для запуска утилиты, а также тестовые файлы.
